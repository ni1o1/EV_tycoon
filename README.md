# EV Tycoon - 充电站经营模拟游戏

一个基于HTML5 Canvas的互动式充电站经营模拟游戏，玩家需要在100天内经营管理自己的充电站，应对各种挑战并实现盈利。

## 游戏特色

- **真实模拟**: 基于真实分时电价和天气系统
- **策略经营**: 平衡设备投资、电价设置和运营成本
- **多样挑战**: 9种不同的游戏模式，每种都有独特的挑战
- **实时数据**: 动态图表显示能源使用情况和收益
- **贷款系统**: 提供等额本息和等额本金两种贷款方式
- **随机事件**: 20种突发事件和20种长期事件增加游戏变数

## 游戏玩法

### 基础设备
- **慢充桩**: 7kW功率，成本较低，适合基础运营
- **快充桩**: 30kW功率，充电速度快，吸引更多客户
- **光伏板**: 10kW发电功率，白天发电降低用电成本
- **储能电池**: 100kWh容量，可储存光伏电力或低价电网电力
- **配电设备**: 增加20kW电网容量上限，避免过载

### 经营策略
- **电价设置**: 在$0.5-$3.0之间自由调整，平衡客户接受度和收益
- **分时电价**: 不同城市和时段电价不同，需合理安排储能充电
- **天气影响**: 光伏效率受天气影响，雨天效率大幅下降
- **客流管理**: 不同地段客流模式不同，需针对性配置设备

### 挑战模式
1. **标准模式**: 平衡体验，适合新手
2. **超充狂热**: 禁止使用慢充，专注快充运营
3. **离网挑战**: 只能使用光伏和储能，不能从电网购电
4. **鬼城危机**: 客流量减半，考验生存能力
5. **豪华地段**: 租金昂贵，需要更高收益
6. **限电模式**: 电网容量紧张，容易过载
7. **恶性通胀**: 设备价格快速上涨
8. **永恒雨季**: 光伏基本失效
9. **高利贷**: 每3天需支付高额租金

## 技术特点

- **纯前端实现**: 基于HTML5 Canvas和原生JavaScript
- **响应式设计**: 适配各种屏幕尺寸，支持移动设备
- **实时渲染**: 动态天气效果和车辆充电动画
- **数据可视化**: 使用ECharts显示24小时能源使用曲线
- **性能优化**: 高效的碰撞检测和渲染循环

## 如何运行

1. 克隆或下载项目到本地
2. 直接在浏览器中打开 `docs/index.html`
3. 开始你的充电站经营之旅！

## 游戏目标

- **短期目标**: 每日盈利，按时支付租金
- **中期目标**: 扩大规模，优化能源配置
- **长期目标**: 存活100天，成为充电站经营大师

## 开发说明

游戏使用纯前端技术开发，无需后端支持。所有数据都在浏览器本地运行，无需网络连接即可游戏。

### 核心模块
- **游戏循环**: 10分钟为单位的实时模拟
- **能源管理**: 光伏、储能、电网的智能调度
- **经济系统**: 动态价格、成本计算和收益统计
- **事件系统**: 随机事件和长期状态效果
- **UI渲染**: Canvas绘制和DOM更新
